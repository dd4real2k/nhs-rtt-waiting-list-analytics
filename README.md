# NHS RTT Waiting List Intelligence

## Project Overview
This project analyses NHS Referral to Treatment (RTT) waiting times using publicly available NHS England data. The aim is to identify waiting list trends, compare provider performance, detect data quality issues, and forecast future service pressure.

## Business Problem
NHS Trusts need timely and reliable intelligence to understand elective care backlogs, monitor performance, and support operational planning. This project demonstrates how SQL, Python, machine learning, and dashboards can be used to turn raw healthcare data into actionable insights.

## Key Skills Demonstrated
- SQL data modelling and complex queries
- Python data cleaning and transformation
- Data quality validation
- Exploratory data analysis
- Healthcare KPI reporting
- Forecasting and machine learning fundamentals
- Dashboard development
- Reproducible analytical workflows

## Dataset
The project uses publicly available NHS England Referral to Treatment waiting times data.
